Add Subject Area

Screen Description

The Add Subject Area screen provides you with interface for adding a new subject area.


Details

  • Academic Session
    • The academic session for which you are adding a new subject area
    • Can be changed from the Academic Sessions screen
  • Abbreviation
    • Subject area abbreviation
  • Short Title
    • Short title of the new subject area
  • Long Title
    • Long title of the new subject area
  • External ID
    • External ID of the subject area (optional)
  • Department
    • Select an existing department to which the subject area belongs (a subject area has to belong to a department)
      • A new department can be added in the Add Department screen - to get there, click on Departments in the Administration section of the left hand side menu
  • Schedule Book Only
    • Check if the subject area is not expected to have any classes associated with it for this academic session
  • Pseudo
    • Check if the subject area is being created for some other purposes than having regular classes (such as CAND to represent that a student is a candidate for graduation)


Operations

  • Save (ALT+S)
    • Save the new subject area and go back to the Subject Areas screen
  • Back (ALT+B)
    • Go back to the Subject Areas screen without saving any changes
